# TrackSense DEM (SRTM3 HGT)
Paquetes de datos de elevación en formato `.hgt` (SRTM3, ~1201x1201) para instalación offline en la app TrackSense.
## Regiones (ZIP)
- Iberia (Península + Portugal) + corona
- Canarias
- Baleares
## Uso
Estos archivos se publican como *Release assets* para que la app pueda descargarlos por HTTPS e instalarlos automáticamente en una carpeta `dem/`.
